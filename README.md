# How to use 
# 1. Normal scan (No live check)
# uuf.sh example.com

# 2. Custom output file
# uuf.sh example.com -o myurls.txt

# 3. Only parameter URLs
# uuf.sh example.com --params

# 4. Only parameter URLs + live URLs
# uuf.sh example.com --params --live -o liveparams.txt

# 5. Full scan + only live URLs
# uuf.sh example.com --live -o live.txt
